Deligatr Assistant — test harness

Configure a mock GHL context, then launch the full-screen chatbot (it takes over the page — reload to come back here). Serve this folder with python3 -m http.server — opening via file:// can break fetch in some browsers.

URL parsing test: leave the mock locationId blank and open test.html?locationId=ABC123&mock=… — the app must pick ABC123 from the URL (check the [dgtr] send console line).
Round-trip test: ask “how do I add a team member?” and confirm a real reply renders.
Plan nudge: set tier for your mock location to DeliHub Free in Supabase, ask for a 3rd user, expect the DeliHub Pro nudge.
Escalation: click “Talk to a human”, then verify the Slack message + GHL task.
To use a local mock endpoint instead of the interceptor, paste its URL (e.g. http://localhost:5678/webhook/deligatr-assistant) into the endpoint field.
